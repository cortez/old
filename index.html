Note

Using a via server and operational transformation, Note is a simple yet powerful plain text collaboration tool, where changes to each generated page are updated in real-time from any device.

The website is currently live at note.cortez.link.

COVID-19 Resources

This project was created to allow me to practice fetching data from APIs using JavaScript and display statistics regarding COVID-19 around the world, along with a number of pandemic-related resources, from CDC websites to small business resources.

Along with an embedded interactive map from Esri, a live coronavirus-related news feed is displayed on the right, fetching from a web search. The website is currently live at covid.cortez.link.

World Places

This project was an online travel brand with the complimentary Twitter account @WorldPlaces. The purpose of this account was to grow an engaging audience interested in travel content to the account's eventual 350K total followers.

Building the account allowed me to talk to and work with talented photographers and videographers, many of which were featured on the account, before it was sold in 2020 to a private buyer.

Unlikely Media

Unlikely Media was a network of Instagram accounts to curate and share user-submitted content, which collectively garnered tens of millions of video views.

Creating and building these accounts taught me the fundamentals of online virality and the role of social media for businesses, allowing me to work with brands such as Goat Case and Fashion Nova among others to advertise various products. Unlikely Media was sold in 2019 to Daily Epic Vids.
